MoneyMaker is a gamified Pomodoro timer and real-time earnings tracker designed for productivity and satisfying visual feedback.

## Features
- **Real-time Earnings Ticker**: Watch your earnings grow as you work based on your monthly salary.
- **The Grimoire**: A multi-page recipe book that functions as a Pomodoro timer.
- **Interactive ASMR Preparation**: Engaging preparation phase before the timer starts.
- **Immersive Environment**: "Salon de Th√©" environment with dynamic day/night logic and interactive elements (Oven, Jar).
- **Physics-simulated Jar**: A visual representation of your progress with realistic physics for coins and bills.
- **Minimalist Widget Mode**: A compact, always-on-top view for tracking your gains while you work.

## Technology Stack
- **Framework**: Electron + React + Vite
- **Styling**: Vanilla CSS + Tailwind CSS
- **State Management**: Zustand
- **Physics Engine**: Matter.js
- **Icons**: Lucide React

## Development
```bash
npm install
npm run dev
```

## Description
MoneyMaker transforms the mundane task of time tracking into an immersive experience. Bake pastries while you earn, fill your virtual jar with currency reflecting your real-world income, and enjoy the cozy atmosphere of your personal tea room.
